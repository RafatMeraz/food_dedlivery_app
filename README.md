# Food delivery app

A Flutter application from which users can visites menus and purchase foods from a specific restaurant.

## Features 
  - Show Menus
  - Search Foods
  - Book a table
  - Order foods
  - Track a order stage
  - User profile management
  - Sign in 
  - Sign up
  
  
## Tachonology
    - Flutter (Mobile framework)
    - Laravel Lumen (REST API microservice)
